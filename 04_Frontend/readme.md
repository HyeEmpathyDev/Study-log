## 🚀 프론트엔드

### 💡 이걸 정리하게 된 이유

내가 뭘 배우고 있는지 모르겠어서..
프론트엔드가 뭔지도 모르는데 html, css를 배우려니?

### ✅ 프론트엔드란?

사용자가 실제로 보고, 클릭하고, 입력하는 **웹사이트의 겉모습**을 만드는 영역
웹페이지의 디자인, 구성, 동작 등을 담당

### ✅ 프론트엔드에서 사용하는 주요 언어

| 언어           | 설명                          |
| -------------- | ----------------------------- |
| **HTML**       | 웹 페이지의 구조(뼈대)를 정의 |
| **CSS**        | 디자인과 스타일을 설정        |
| **JavaScript** | 동작과 인터랙션을 구현        |

<br>

나는 디자인과 화면 구성에 관심이 없는 것 같다. 어렵다.
